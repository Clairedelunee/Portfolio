Koti'fight

Koti'fight est un projet de web application pour lequel il nous a été demandé de créer un média spécialisé pour les 18-25ans avec la problématique suivante :
"Comment intéresser les jeunes à du contenu informationnel autrement que sur les plateformes qu’ils consultent habituellement - TikTok, Instagram ?"
Nous avons donc réalisé ce média autour des fakes news. Le concept ? Une thématique qui dure 2 semaines, 1 questions par jour et deux équipes qui s'affrontent pour avoir le plus de Koti'point!
Malheureusement notre projet ne peut pas être hébergé gratuitement et est donc indisponible sur internet.
