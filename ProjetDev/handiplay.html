Handiplay

Un jeu, une contrainte ! Ce projet avait pour but de nous faire développer notre premier jeu vidéo sur Unity en prenant compte d'un ou plusieurs handicaps. Nous avons développé Thun, en permettant aux personnes souffrantes d'handicaps visuels concernant les couleurs de pouvoir jouer sans difficulté. Toutes les couleurs importantes dans le jeu se matérialisent par une forme qui lui est propre. Ce jeu peut aussi être jouer sans le son, aucune information majeure y est développé par ce biais.
Pour retrouver le jeu : https://amelieperrot.itch.io/thun-project
